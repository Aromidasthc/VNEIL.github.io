# Demo Python — VNEIL-GENESIS

Instrukcje uruchomienia demo Python (Flask):

1. Stwórz wirtualne środowisko i aktywuj je:

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
.venv\Scripts\Activate    # Windows PowerShell
```

2. Zainstaluj zależności:

```bash
pip install -r requirements.txt
```

3. Uruchom aplikację:

```bash
python app.py
# otwórz http://localhost:5000
```
